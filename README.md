# Moneyfy

MoneyFy is a platform that helps you control your spending and achieve your savings goals.

- 💰 Add your incomes 
- 💸 Control your expenses
- 📈 Set your target
- 🎯 Calculate how long it will take you to reach it

## ✨  Technology used

| Frontend (Web) | Backend (API REST) |
| ------- |  ----- |
| JavaScript | NodeJS  |
| React | Express  |
| HTML | MongoDB  |
| SASS | Mongoose  |
| React | 301  |
